## 기여 통계

### Code Contributions

```bash
# 기여자 목록 확인
git log --format='%aN' | sort | uniq -c | sort -rn

# 파일별 기여자
git log --pretty=format:"%an" --name-only | sort | uniq

# 최근 기여자 (30일)
git log --since="30 days ago" --format='%aN' | sort | uniq
```
